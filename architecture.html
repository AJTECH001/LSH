🛡️ LiquidationShieldHook

src/hooks/LiquidationShieldHook.sol
activateShield()external
User registers lending position. Transfers deposit, stores ShieldPosition, emits HealthCheckRequested for Reactive to subscribe.
deactivateShield()external
User exits protection and retrieves remaining deposit balance.
topUpDeposit()external
Adds more protection funds to an existing active shield.
updateThreshold()external
Updates the user's health factor threshold. Re-emits HealthCheckRequested so Reactive Network picks up the new value.
executeProtection()external
Called exclusively by CallbackReceiver. Validates cooldown, deducts deposit, charges 0.5% fee, approves and repays lending protocol.
_afterInitialize()hook
Called by Uniswap v4 after a pool is created. Registers the pool for shield-related swap tracking.
_beforeSwap()hook
Grants a 50% LP fee discount when hookData encodes an active protected user address. Uses LPFeeLibrary.OVERRIDE_FEE_FLAG.
_afterSwap()hook
Emits ProtectionSwapLogged when a shield-tagged swap completes. Does not execute protection — that is handled by executeProtection().
withdrawFees()admin
Owner withdraws accumulated fees for a single specified token.
withdrawAllFees()admin
Owner withdraws all accumulated protocol fees across every collected token in one call.
setCallbackReceiver() / setMinDeposit()admin
Owner updates the authorized callback receiver address or the minimum deposit floor. Both revert with NotAuthorized for non-owners.

🔍 HealthFactorMonitor

src/reactive/HealthFactorMonitor.sol
init()external
Called once after deployment by the owner. Subscribes to HealthCheckRequested events from the hook contract on the origin chain. Gated behind !vm — only executes on the Reactive Network, not inside ReactVM.
react()external
Entry point called by the Reactive Network on every subscribed event. Gated by vmOnly. Routes to _handleHealthCheckRequest or _handleLendingEvent based on topic_0.
_handleHealthCheckRequest()internal
Registers a new user for monitoring when HealthCheckRequested is detected. Calls _subscribeToLendingEvents.
_handleLendingEvent()internal
Processes Borrow/Repay/Liquidation events. Estimates health factor and triggers _triggerProtection if below threshold.
_estimateHealthFactor()internal
Simplified HF estimation from log data. Production version queries oracle prices for precise calculation.
_triggerProtection()internal
Emits Callback event with encoded executeProtection payload. Reactive relayer delivers this cross-chain.
_subscribeToLendingEvents()internal
Subscribes to Borrow, Repay and LiquidationCall events on the lending pool via ISubscriptionService.
addMonitoredPosition()admin
Owner can manually register a position for monitoring without waiting for the hook's HealthCheckRequested event.

📡 CallbackReceiver

src/reactive/CallbackReceiver.sol
receiveCallback()external
Main entry for Reactive Network relayer. Decodes raw payload, emits CallbackReceived, then forwards to hook via low-level call.
triggerProtection()external
Alternative typed entry point — calls executeProtection() directly without raw payload decoding.
addAuthorizedCaller()admin
Whitelists an address (e.g. the Reactive callback proxy) that is allowed to call receiveCallback / triggerProtection.
removeAuthorizedCaller()admin
Removes a previously whitelisted caller from the authorized set.
setHook()admin
Owner updates the hook contract address that callbacks are forwarded to. Reverts with NotAuthorized for non-owners.

📋 Key Data Structures & Events

LiquidationShieldHook.sol
ShieldPosition struct
user · originChainId · lendingPool · debtToken · collateralToken · healthThreshold (1e18 scale) · depositBalance · protectionFee (bps) · lastTriggered (timestamp) · isActive.
ShieldActivated event
Emitted on activateShield(). Indexed: user, originChainId. Data: lendingPool, healthThreshold, depositAmount.
ShieldDeactivated event
Emitted on deactivateShield(). Indexed: user. Data: refundAmount (0 if deposit was fully consumed).
HealthCheckRequested event
Emitted on activateShield() and updateThreshold(). All four params indexed. Reactive Network picks this up to start monitoring.
ShieldTriggered event
Emitted after successful executeProtection(). Fields: user (indexed), originChainId (indexed), healthFactorBefore, repayAmount, newHealthFactor (0 = calculated off-chain).
ProtectionSwapLogged event
Emitted in _afterSwap when a shield-tagged swap passes through the pool.
ProtectionSwapLogged event
Emitted in _afterSwap when hookData encodes an active protected user. Fields: user (indexed), originChainId (indexed), swapAmount.
Constants & bounds
COOLDOWN_PERIOD = 5 minutes  ·  DEFAULT_FEE_BPS = 50 (0.5%)  ·  threshold range: 1.0e18 – 2.0e18
